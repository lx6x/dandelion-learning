knife4j-openapi3-spring-boot-starter 是 Knife4j 的 Spring Boot Starter 模块，它提供了在 Spring Boot 应用程序中集成
Knife4j 的能力。它基于 OpenAPI 3.0 规范，可以自动生成 API 文档，并提供了丰富的 UI 界面和交互功能，使得 API 文档的查看和测试更加方便。

使用 knife4j-openapi3-spring-boot-starter，你可以在 Spring Boot 应用程序中添加相关的注解，如 @Api、@ApiOperation 等，来描述和标记你的
API 接口。然后，Knife4j 将自动扫描这些注解，并生成相应的 API 文档页面。

通过 knife4j-openapi3-spring-boot-starter，你可以在浏览器中访问生成的 API 文档页面，查看接口详情、测试接口调用等。

knife4j-spring-ui 是 Knife4j 的前端 UI 模块，它提供了 Knife4j 的前端界面和交互功能，可以单独作为一个独立的前端项目使用。

如果你想自定义 Knife4j 的前端界面，或者将 Knife4j 的 UI 集成到自己的前端项目中，你可以使用 knife4j-spring-ui
模块。它提供了一组静态资源（HTML、CSS、JavaScript）和模板文件，可以根据需要进行自定义和集成。

knife4j-openapi3-spring-boot-starter 是用于在 Spring Boot 应用程序中集成 Knife4j 的模块，提供了自动生成 API 文档的功能；而
knife4j-spring-ui 是 Knife4j 的前端 UI 模块，用于自定义和集成 Knife4j 的前端界面。
